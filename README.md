## üèóÔ∏è Architectural Framework
The system is built on a decoupled architecture, separating the **Stochastic Data Factory** from the **Sovereign Intelligence UI**.

### 1. Stochastic Data Factory (`run_pipeline.py`)
Generates 365 days of temporal telemetry across 15 global nodes (Mumbai, Suez, Rotterdam, etc.). Unlike static data, this engine uses **Geometric Brownian Motion (GBM)** to simulate realistic market drift:
$$dS_t = \mu S_t dt + \sigma S_t dW_t$$



### 2. Intelligence Kernel (`app.py`)
A glassmorphic executive dashboard that provides:
* **Dynamic VaR Calculation:** Value at Risk adjusted by systemic entropy and shock factors.
* **Stress Scenario Injection:** Simulated Suez Canal blockades and Cyber Attacks.
* **Pareto Optimization:** Solving for the optimal balance between Profit and ESG (Sustainability) metrics.
